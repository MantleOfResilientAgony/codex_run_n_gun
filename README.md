# codex_run_n_gun

This repository contains a minimal side-scrolling "run and gun" game implemented with [Pygame](https://www.pygame.org/).

## Requirements

- Python 3.8+
- Pygame

Install dependencies with pip:

```bash
pip install -r requirements.txt
```

## Running the Game

Run the main module:

```bash
python main.py
```

Use the left and right arrow keys to move, the up arrow to jump, and the space bar to shoot. If an enemy touches the player, the game ends.
